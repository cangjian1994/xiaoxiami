## 🔧 OpenClaw 核心配置

### 1. openclaw.json 配置

```json
{
  "channels": {
    "feishu": {
      "type": "feishu",
      "appId": "cli_xxxxxxxxxxxxx",
      "appSecret": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
      "appTicket": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
    }
  }
}
```

### 2. 飞书应用配置步骤

#### 步骤 1: 创建应用
1. 访问 https://open.feishu.cn/app
2. 创建企业自建应用
3. 填写应用名称、图标

#### 步骤 2: 配置权限
必需权限：
- 即时消息
- 机器人  
- 云文档
- 通讯录

#### 步骤 3: 获取凭证
- App ID
- App Secret

#### 步骤 4: 事件订阅
1. 开启事件订阅
2. 配置订阅地址
3. 获取 App Ticket

#### 步骤 5: 发布应用
创建版本并发布

### 3. Gateway 配置

```bash
# 启动
openclaw gateway start

# 查看状态
openclaw gateway status

# 重启
openclaw gateway restart
```
